# StockPortfolioAnalyzer

## 簡介
庫存計算機

## 數據爬取來源
* [yfinance](https://pypi.org/project/yfinance/)

## 環境搭建

* 安裝套件
```
pip install PyQt5
```

## 使用方法

* 下載此 Git 存放庫
```
git clone https://github.com/paulkuo123/InventoryCalculator.git
```
* 切換至 InventoryCalculator 資料夾
```
cd InventoryCalculator
```
* 執行 main.py
```
python main.py
```
